### 7.5 验收标准

#### 阶段一验收标准

- [ ] 支持至少2家大模型API调用
- [ ] 模型故障自动降级
- [ ] 对话上下文保持10轮以上
- [ ] 响应时间 < 3秒

#### 阶段二验收标准

- [ ] 课程知识库覆盖80%以上内容
- [ ] RAG回答准确率 > 70%
- [ ] 检索响应时间 < 500ms

#### 阶段三验收标准

- [ ] 支持3种以上Agent类型
- [ ] 工具调用成功率 > 90%
- [ ] 用户意图识别准确率 > 80%

#### 阶段四验收标准

- [ ] 语音识别准确率 > 85%
- [ ] 语音合成自然度可接受
- [ ] 图像分析响应时间 < 5秒

### 8.3 测试用例

```java
@SpringBootTest
class AgentServiceTest {
    
    @Autowired
    private ChatService chatService;
    
    @Test
    void testBasicChat() {
        ChatRequest request = ChatRequest.builder()
                .message("你好，请介绍一下Python")
                .sessionId("test-session-001")
                .build();
        
        ChatResponse response = chatService.chat(request);
        
        assertNotNull(response.getReply());
        assertFalse(response.getReply().contains("mock"));
        assertEquals("test-session-001", response.getSessionId());
    }
    
    @Test
    void testConversationMemory() {
        String sessionId = "test-memory-001";
        
        // 第一轮对话
        chatService.chat(ChatRequest.builder()
                .message("我叫张三")
                .sessionId(sessionId)
                .build());
        
        // 第二轮对话，测试记忆
        ChatResponse response = chatService.chat(ChatRequest.builder()
                .message("我叫什么名字？")
                .sessionId(sessionId)
                .build());
        
        assertTrue(response.getReply().contains("张三"));
    }
    
    @Test
    void testModelFallback() {
        // 模拟主模型故障，测试降级
        when(primaryModel.generate(any())).thenThrow(new RuntimeException());
        
        String response = modelRoutingService.generateWithFallback("测试");
        
        assertNotNull(response);
        verify(fallbackModel).generate(any());
    }
}
```

## 九、快速启动指南

### 9.1 环境准备

```bash
# 1. 启动Milvus向量数据库
docker-compose -f docker-compose-milvus.yml up -d

# 2. 配置环境变量
export WENXIN_API_KEY=your_key
export QWEN_API_KEY=your_key

# 3. 启动服务
cd backend/edu-agent-service
mvn spring-boot:run
```

### 技术架构亮点

#### 微服务架构设计
```
├── 用户服务 (User Service)
├── 课程服务 (Course Service)  
├── 学习服务 (Learning Service)
└── 资源服务 (Resource Service)
```

#### 技术栈选型
- **后端**：Spring Boot 2.7.10 + MyBatis Plus 3.5.3
- **数据库**：MySQL 8.0 + Redis 6.x
- **前端**：Vue3 + Element Plus（预留接口）
- **测试**：JUnit 5 + Mockito
- **部署**：Docker + Nginx反向代理

## 系统架构

### 整体架构图
```
┌─────────────────────────────────────────────────────────────┐
│                        前端层 (Frontend)                     │
│  Vue3 + Element Plus + Axios                               │
├─────────────────────────────────────────────────────────────┤
│                        网关层 (Gateway)                      │
│  Nginx + Spring Cloud Gateway                              │
├─────────────────────────────────────────────────────────────┤
│                        服务层 (Services)                     │
│  User Service  │  Course Service  │  Learning Service      │
├─────────────────────────────────────────────────────────────┤
│                        中间件层 (Middleware)                 │
│  Redis Cache  │  RabbitMQ  │  Elasticsearch                │
├─────────────────────────────────────────────────────────────┤
│                        数据层 (Data Layer)                   │
│  MySQL 8.0  │  MongoDB  │  文件存储                        │
└─────────────────────────────────────────────────────────────┘
```

### 微服务架构
- **用户服务**：负责用户认证、权限管理
- **课程服务**：处理课程创建、管理功能
- **学习服务**：记录学习行为、分析学习数据
- **消息服务**：处理系统通知、实时通信
- **资源服务**：管理教学资源、文件存储

## 数据库设计

### 4.1 核心数据表

#### 4.1.1 用户表 (edu_user)
```sql
CREATE TABLE edu_user (
    user_id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '用户ID',
    username VARCHAR(50) UNIQUE NOT NULL COMMENT '用户名',
    password VARCHAR(100) NOT NULL COMMENT '密码',
    real_name VARCHAR(50) NOT NULL COMMENT '真实姓名',
    phone VARCHAR(20) UNIQUE COMMENT '手机号',
    email VARCHAR(100) COMMENT '邮箱',
    user_type TINYINT NOT NULL COMMENT '用户类型：1-学生 2-教师 3-家长 4-管理员',
    status TINYINT DEFAULT 1 COMMENT '状态：0-禁用 1-启用',
    avatar VARCHAR(200) COMMENT '头像URL',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
);
```

#### 4.1.2 课程表 (edu_course)
```sql
CREATE TABLE edu_course (
    course_id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '课程ID',
    course_name VARCHAR(100) NOT NULL COMMENT '课程名称',
    course_code VARCHAR(50) UNIQUE NOT NULL COMMENT '课程编码',
    description TEXT COMMENT '课程描述',
    teacher_id BIGINT NOT NULL COMMENT '授课教师ID',
    grade_level TINYINT NOT NULL COMMENT '年级',
    subject_id BIGINT NOT NULL COMMENT '学科ID',
    credit DECIMAL(3,1) DEFAULT 0 COMMENT '学分',
    status TINYINT DEFAULT 1 COMMENT '状态：0-停用 1-启用',
    cover_image VARCHAR(200) COMMENT '封面图片',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### 4.1.3 学习记录表 (edu_learning_record)
```sql
CREATE TABLE edu_learning_record (
    record_id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '记录ID',
    student_id BIGINT NOT NULL COMMENT '学生ID',
    course_id BIGINT NOT NULL COMMENT '课程ID',
    lesson_id BIGINT COMMENT '课时ID',
    study_duration INT DEFAULT 0 COMMENT '学习时长（分钟）',
    progress INT DEFAULT 0 COMMENT '学习进度百分比',
    is_completed BOOLEAN DEFAULT FALSE COMMENT '是否完成',
    last_study_time DATETIME COMMENT '最后学习时间',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 4.2 数据库索引策略
```sql
-- 用户表索引
CREATE INDEX idx_user_username ON edu_user(username);
CREATE INDEX idx_user_phone ON edu_user(phone);
CREATE INDEX idx_user_type ON edu_user(user_type);

-- 课程表索引
CREATE INDEX idx_course_teacher ON edu_course(teacher_id);
CREATE INDEX idx_course_grade ON edu_course(grade_level);
CREATE INDEX idx_course_subject ON edu_course(subject_id);

-- 学习记录表索引
CREATE INDEX idx_learning_student ON edu_learning_record(student_id);
CREATE INDEX idx_learning_course ON edu_learning_record(course_id);
CREATE INDEX idx_learning_time ON edu_learning_record(last_study_time);
```

## 部署指南

### 5.1 环境准备
#### 5.1.1 服务器要求
- **操作系统**：CentOS 7.6+ 或 Ubuntu 18.04+
- **内存**：最低4GB，推荐8GB+
- **存储**：最低50GB可用空间
- **网络**：公网IP或内网可访问

#### 5.1.2 软件依赖
```bash
# Java环境
sudo yum install java-1.8.0-openjdk-devel

# MySQL 8.0
sudo yum install mysql-server

# Redis 6.x
sudo yum install redis

# Nginx
sudo yum install nginx
```

### 5.2 应用部署
#### 5.2.1 构建项目
```bash
# 克隆代码
git clone https://github.com/your-org/ai-edu-platform.git
cd ai-edu-platform

# Maven构建
mvn clean package -DskipTests

# 生成jar包位置
ls target/*.jar
```

#### 5.2.2 配置文件
```yaml
# application-prod.yml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/edu_platform?useUnicode=true&characterEncoding=utf8
    username: ${DB_USERNAME}
    password: ${DB_PASSWORD}
  
  redis:
    host: localhost
    port: 6379
    password: ${REDIS_PASSWORD}

logging:
  level:
    com.chinaunicom.edu: INFO
  file:
    name: ./logs/application.log
```

#### 5.2.3 启动应用
```bash
# 后台运行
nohup java -jar target/ai-edu-platform-1.0.0.jar \
  --spring.profiles.active=prod \
  --server.port=8080 > app.log 2>&1 &

# 查看进程
ps -ef | grep ai-edu-platform
```

### 5.3 Nginx配置
```nginx
server {
    listen 80;
    server_name edu.example.com;
    
    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
    
    location /api/ {
        proxy_pass http://localhost:8080/api/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 8.2 缓存策略
```java
// Redis缓存配置
@Configuration
@EnableCaching
public class RedisConfig {
    
    @Bean
    public RedisTemplate<String, Object> redisTemplate(RedisConnectionFactory factory) {
        RedisTemplate<String, Object> template = new RedisTemplate<>();
        template.setConnectionFactory(factory);
        return template;
    }
}
```

### 8.3 前端优化
```javascript
// API请求优化
const api = axios.create({
    baseURL: '/api',
    timeout: 10000,
    headers: {
        'Cache-Control': 'no-cache'
    }
});

// 请求拦截器
api.interceptors.request.use(config => {
    const token = localStorage.getItem('token');
    if (token) {
        config.headers.Authorization = `Bearer ${token}`;
    }
    return config;
});
```

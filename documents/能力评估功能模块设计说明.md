# 能力评估功能模块设计说明

## 1. 模块概述

能力评估模块是智慧教育平台的核心功能之一，旨在通过科学的评估体系帮助学生了解自身学习水平，为教师提供教学参考，为系统推荐个性化学习路径。

## 2. 核心功能特性

### 2.1 多维度评估体系
- **多样化评估类型**：入学测试、阶段性测试、期末测试、专项能力测试
- **灵活题型支持**：单选题、多选题、判断题、填空题、简答题
- **分级难度设置**：简单、中等、困难三级难度体系
- **知识点标签管理**：支持按知识点分类和检索

### 2.2 智能评估机制
- **自动评分系统**：客观题自动判分，主观题人工批改
- **实时能力分析**：基于答题情况生成能力画像
- **个性化推荐**：根据评估结果推荐学习内容和路径
- **防作弊机制**：限时作答、题目乱序、IP限制等

### 2.3 数据统计分析
- **个人学习轨迹**：记录历次评估成绩和发展趋势
- **群体数据分析**：班级、年级整体水平分析
- **知识点掌握度**：各知识点的掌握情况统计
- **成长趋势预测**：基于历史数据预测学习发展趋势

## 3. 系统架构设计

### 3.1 实体类设计

#### Assessment（评估）
```java
@Data
@TableName("edu_assessment")
public class Assessment {
    private Long assessmentId;      // 评估ID
    private String assessmentName;  // 评估名称
    private String assessmentCode;  // 评估编码
    private Long subjectId;         // 学科ID
    private Integer gradeLevel;     // 适用年级
    private Integer assessmentType; // 评估类型
    private String description;     // 评估描述
    private BigDecimal totalScore;  // 总分
    private BigDecimal passingScore;// 及格分数
    private Integer duration;       // 考试时长
    private Integer questionCount;  // 题目数量
    private Integer status;         // 状态
    private Long creatorId;         // 创建者ID
    private LocalDateTime startTime;// 开始时间
    private LocalDateTime endTime;  // 结束时间
    private Integer allowRepeat;    // 是否允许重复参加
    private Integer repeatInterval; // 重考间隔
}
```

#### AssessmentQuestion（评估题目）
```java
@Data
@TableName("edu_assessment_question")
public class AssessmentQuestion {
    private Long questionId;        // 题目ID
    private Long assessmentId;      // 评估ID
    private Integer questionOrder;  // 题目序号
    private Integer questionType;   // 题目类型
    private String content;         // 题目内容
    private String options;         // 题目选项(JSON)
    private String correctAnswer;   // 正确答案
    private BigDecimal score;       // 题目分数
    private Integer difficulty;     // 难度等级
    private String tags;            // 知识点标签
    private String analysis;        // 解析说明
}
```

#### UserAssessment（用户评估结果）
```java
@Data
@TableName("edu_user_assessment")
public class UserAssessment {
    private Long resultId;          // 结果ID
    private Long userId;            // 用户ID
    private Long assessmentId;      // 评估ID
    private LocalDateTime startTime;// 开始时间
    private LocalDateTime endTime;  // 结束时间
    private Integer timeSpent;      // 用时(秒)
    private BigDecimal totalScore;  // 总得分
    private BigDecimal scorePercentage;// 得分百分比
    private Integer isPassed;       // 是否及格
    private String answerDetails;   // 答题详情(JSON)
    private Integer status;         // 评估状态
    private String teacherComment;  // 教师评语
    private String capabilityReport;// 能力分析报告(JSON)
    private String recommendedPath; // 推荐学习路径
}
```

### 3.2 服务层设计

#### AssessmentService接口
```java
public interface AssessmentService extends IService<Assessment> {
    // 获取可参加的评估列表
    List<Assessment> getAvailableAssessments(Long userId);
    
    // 根据学科获取评估列表
    List<Assessment> getAssessmentsBySubject(Long subjectId);
    
    // 获取评估详情（包含题目）
    Map<String, Object> getAssessmentDetail(Long assessmentId);
    
    // 开始评估
    UserAssessment startAssessment(Long userId, Long assessmentId);
    
    // 提交答案
    UserAssessment submitAnswer(Long userId, Long assessmentId, Map<Integer, String> answers);
    
    // 获取用户评估历史
    List<UserAssessment> getUserAssessmentHistory(Long userId);
    
    // 生成能力分析报告
    Map<String, Object> generateCapabilityReport(Long userAssessmentId);
    
    // 检查用户权限
    boolean canUserTakeAssessment(Long userId, Long assessmentId);
}
```

### 3.3 控制器层设计

#### 主要API接口
- `GET /api/assessments/available` - 获取可参加的评估列表
- `GET /api/assessments/{assessmentId}` - 获取评估详情
- `POST /api/assessments/{assessmentId}/start` - 开始评估
- `POST /api/assessments/{assessmentId}/submit` - 提交答案
- `GET /api/assessments/history/{userId}` - 获取评估历史
- `GET /api/assessments/statistics/{userId}` - 获取统计信息
- `GET /api/assessments/report/{userAssessmentId}` - 生成能力报告

## 4. 数据库设计

### 4.1 表结构说明

#### edu_assessment（评估表）
存储各种类型的评估信息，包括基本信息、时间安排、参与规则等。

#### edu_assessment_question（评估题目表）
存储评估的具体题目内容，支持多种题型和配置选项。

#### edu_user_assessment（用户评估结果表）
记录用户的每次评估参与情况和结果，包含详细的成绩分析。

#### edu_assessment_statistics（评估统计表）
存储评估的整体统计数据，便于后续分析和报表生成。

### 4.2 关键索引设计
